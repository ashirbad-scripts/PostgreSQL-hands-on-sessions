# PostgreSQL-hands-on-sessions
## Overview
This repository is dedicated to practicing PostgreSQL queries.  
It acts as a personal workspace where I continuously write, test, and refine SQL queries while learning and experimenting with different concepts.

The practice is session-based, with queries added and improved over time.

## Goals
- Build strong PostgreSQL query-writing skills  
- Improve logical thinking for data retrieval and analysis  
- Maintain a consistent SQL practice habit  
- Create a reusable reference for common and advanced SQL patterns  

## What It Covers
- Basic SQL queries and filtering
- CTE's
- Window functions
- String Operations
- Date Time Operations

## Update Frequency
Updated regularly as part of ongoing SQL practice.  
Queries may be added, modified, or optimized over time.
